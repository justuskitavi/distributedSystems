
## Requirements
- Python 3.x
- Operating System: Linux, Windows, or MacOS
- No external libraries required (uses built-in `xmlrpc` library)

## How to Run
1. Open a terminal and navigate to the `RPC` folder.
2. Start the RPC server:
    ```bash
    python3 server.py

3. in a second terminal, start the RPC client:
    ```bash
    python3 client.py
    
4. Follow the prompts
