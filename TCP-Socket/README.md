
## Requirements
- Python 3.x
- Operating System: Linux, Windows, or MacOS
- No external libraries required

## How to Run
1. Open a terminal and navigate to the `TCP-Sockets` folder.
2. Start the server:
    ```bash
    python server.py
3. in a second terminal, start the client:
    ```bash
    python3 client.py
    
4. Now type a message thats gonna be echoed by the server
5. Press Ctrl + C to terminate the connection
